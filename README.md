EasyFramework 
Client 为 Unity游戏框架
  集成了 HyBridCLR 将 Assembly-CSharp 作为热更程序集,简单易用
  也包含了
    资源管理(AssetsMgr)
    文件管理(FileMgr)
    配置管理(ConfigMgr)
    事件管理(EventMgr)
    协程管理(CoroutineMgr)
    有限状态机(FMSMachine)
    多语言(I18NMgr)
    输入管理(InputMgr)
    UI管理(UIMgr)
    ...功能
  也有便于管理的启动加载流程

Mod为配置表转换工具
  可将Excel表转换为 Flatbuffer,ProtoBuffer,JS,TS,JSON
